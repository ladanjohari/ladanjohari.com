(Essay) Time planning tools: From presenting raw information to synthesizing candidate solutions

I wrote an essay that provides a taxonomy of time planning tools. For each

  • Type A: Tools that present raw information with no assistance and expect the user to find the solution
  • Type B: Tools that find a solution given a high-level criteria from the user
  • Type C: Tools that proactively predict the intent of the user and provide solutions
You can read the document on Google Docs.
